# BuilderLayout

## 导入

```ts
import { BuilderLayout } from '@visactor/vbi-react/components'
```

## 签名

```ts
BuilderLayout(props: BuilderLayoutProps)
```

## 说明

提供标准化三栏/双栏构建器布局容器。

## 推荐先看

- [组件化布局](../examples/layoutWithComponents)
- [vbi-react Starter](../examples/vbi-react-starter)
