# FilterPanel

## 导入

```ts
import { FilterPanel } from '@visactor/vbi-react/components'
```

## 签名

```ts
FilterPanel(props: FilterPanelProps)
```

## 说明

提供 Where / Having 过滤条件的新增、编辑、删除与分组展示能力。
